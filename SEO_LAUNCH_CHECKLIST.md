# Free InstaSaver SEO launch checklist

Use this after buying the domain and publishing the site.

## Domain and indexing

1. Point the domain to the live hosting URL. For Render, add the domain in the Render Custom Domains screen first, then copy the DNS records Render gives you.
2. Set the production environment variable if your host supports it:

   ```text
   SITE_URL=https://freeinstasaver.com
   ```

3. Confirm these URLs load:

   ```text
   https://freeinstasaver.com/
   https://freeinstasaver.com/instagram-reels-downloader
   https://freeinstasaver.com/instagram-video-downloader
   https://freeinstasaver.com/instagram-story-downloader
   https://freeinstasaver.com/instagram-audio-downloader
   https://freeinstasaver.com/robots.txt
   https://freeinstasaver.com/sitemap.xml
   ```

4. Add the domain to Google Search Console.
5. Submit:

   ```text
   https://freeinstasaver.com/sitemap.xml
   ```

## Pages created for SEO

- `/` — main Instagram downloader page
- `/instagram-reels-downloader` — Reels keyword page
- `/instagram-video-downloader` — public post/video keyword page
- `/instagram-story-downloader` — story keyword page
- `/instagram-audio-downloader` — audio keyword page

## Ad network setup

When Adsterra or another ad network gives you an `ads.txt` line, add it before applying for ads approval.

Do not invent `ads.txt` content. Use the exact line from the ad network dashboard.

## Ranking reality

The site now has the technical SEO base: titles, descriptions, canonical URLs, sitemap, robots, structured data, internal links, FAQ content, and keyword landing pages. Ranking will still depend on domain age, useful content, page speed, backlinks, search demand, and whether search engines consider the downloader content safe and legitimate.
